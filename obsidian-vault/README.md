## Cómo usar este vault

1. **Abrir con Obsidian** — Abre esta carpeta como vault en Obsidian Desktop
2. **Vista de grafo** — Usa Cmd+G para ver el grafo de conexiones entre notas
3. **Punto de entrada** — Comienza siempre por `00-INDEX/MOC-Principal.md`

### O bien, abrir el Graphity interactivo:
👉 Abrir `assets/graphity.html` en tu navegador para ver el grafo visual del proyecto
